HTML/CSS

/full-target.html?target=[TARGET]&reload_interval=[SECONDS]
target
which target to display
reload_interval
reload whole page after specified seconds (default: 0, no reload); minimum is 10 seconds
Example: /full-target.html?target=devytc&reload_interval=60

Rest API

/api/v1/targets/[TARGET]/full
Example: /api/v1/targets/devytc/full
